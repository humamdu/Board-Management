# Task & Notes Management Platform

A Laravel 12, Sanctum, Reverb, Vue 3, TypeScript, Pinia, and TailwindCSS Kanban platform for boards, lists, task/note cards, tags, priorities, checklists, comments, attachments, search, filters, notifications-ready activity logging, and responsive dashboards.

## Highlights
- Multi-workspace and board architecture with board membership roles.
- Kanban lists and drag-and-drop-ready card move API.
- Cards support task/note types, priorities, due dates, assignees, tags, comments, checklists, attachments, and activity auditing.
- Search across title, description, comments, and tags with composable filters.
- Docker Compose for PHP, MySQL 8, and Reverb.
